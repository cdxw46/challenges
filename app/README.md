# FocusFlow — Productivity Dashboard

A beautiful, dynamic productivity dashboard built with React, Tailwind CSS, and Framer Motion. Features a Pomodoro timer, task manager, focus statistics, and motivational quotes.

## Features

- **Pomodoro Timer** — Focus, Short Break, and Long Break modes with animated circular progress
- **Task Manager** — Add, complete, delete tasks with priority levels and categories
- **Focus Statistics** — Track sessions, hours, and weekly focus with interactive charts
- **Dark Mode** — Automatic system detection with manual toggle
- **Motivational Quotes** — Auto-rotating quotes to keep you inspired
- **Persistent Data** — Tasks and sessions saved to localStorage
- **Smooth Animations** — Framer Motion powered transitions throughout

## Tech Stack

- **React 19** + Vite
- **Tailwind CSS v4**
- **Framer Motion** — Animations
- **Recharts** — Charts
- **Lucide React** — Icons

## Getting Started

```bash
cd app
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```
